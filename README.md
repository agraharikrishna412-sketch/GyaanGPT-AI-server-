# GyaanGPT Server

## Quick Start
1. Clone repo
2. `npm install`
3. Copy `.env.example` to `.env` and fill API keys
4. Run server:
   - Dev: `npm run dev`
   - Prod: `npm start`

## Features
- AI Assistant (Llama 3.1) for full app management
- Connectors: DeepSeek, Claude, GPT, Gemini backup
- Voice & Translation support
- Exam Predictor & QBank
- Smart Board / Animated Lessons

## Environment Variables
- PORT, NODE_ENV, API_KEYS, MONGO_URI, STORAGE_PATH, JWT_SECRET
